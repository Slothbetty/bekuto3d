SVG to STL API Example

Convert SVG to STL

Upload an SVG file to convert it to a 3D STL model using the bekuto3d API.

API Information

Endpoint: POST http://localhost:3001/api/svg-to-stl

Content-Type: multipart/form-data

Field Name: file (or any name)

Example JavaScript Usage:

const formData = new FormData();
formData.append('file', svgFile);

const response = await fetch('http://localhost:3001/api/svg-to-stl', {
    method: 'POST',
    body: formData
});

if (response.ok) {
    const result = await response.json();
    if (result.success) {
        // Download the STL file using the provided URL
        const a = document.createElement('a');
        a.href = result.downloadUrl;
        a.download = result.filename;
        a.click();
        
        console.log(`File size: ${result.fileSize} bytes`);
        console.log(`Shapes processed: ${result.shapes}`);
    }
}
